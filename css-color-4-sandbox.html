CSS Color 4 Sandbox

Entdecken Sie die neuen Farbmodelle und Color-Spaces von CSS Color 4 im Vergleich zu traditionellen Methoden

Hinweis: Für die beste Darstellung verwenden Sie einen modernen Browser mit Wide-Gamut Display Support (Safari, Chrome 111+, Firefox 113+)

LAB Color Space - Perzeptuelle Gleichmäßigkeit

LAB basiert auf der menschlichen Wahrnehmung und bietet gleichmäßige Farbabstände.

LAB Rot
lab(70% 20 30)
L=70% (Helligkeit), a=20 (Grün→Rot), b=30 (Blau→Gelb)
LAB Grün
lab(80% -30 40)
Negative a-Werte = Grün, positive b-Werte = Gelb
LAB Blau
lab(60% 10 -40)
Negative b-Werte = Blau, positive a-Werte = Rot
LAB Vorteile: Perzeptuelle Gleichmäßigkeit, geräteunabhängig, große Farbpalette

LAB vs. RGB Vergleich

LAB: lab(50% 50 50)
RGB: rgb(170, 90, 60)

LCH Color Space - Zylindrische LAB-Darstellung

LCH verwendet Helligkeit (L), Chroma (C) und Hue (H) - intuitivere Farbmanipulation.

LCH Rot
lch(70% 50 0)
0° = Rot, 120° = Grün, 240° = Blau
LCH Grün
lch(70% 50 120)
Chroma = Farbintensität (0 = Grau)
LCH Blau
lch(70% 50 240)
Hue-Rotation ist intuitiv und vorhersagbar

Chroma-Intensität Demo

Von links nach rechts: Chroma 0, 20, 40, 60, 80

OKLCH Color Space - Verbesserte Perzeption

OKLCH ist eine verbesserte Version von LCH mit besserer perzeptueller Gleichmäßigkeit.

OKLCH Rot
oklch(0.7 0.15 0)
L: 0-1, C: 0-0.4+, H: 0-360°
OKLCH Grün
oklch(0.7 0.15 120)
Bessere Farbwahrnehmung als LCH
OKLCH Blau
oklch(0.7 0.15 240)
Empfohlen für moderne Designs
OKLCH Besonderheiten: Verbesserte perzeptuelle Gleichmäßigkeit, optimiert für moderne Displays, bessere Farb-Interpolation

OKLAB Color Space - Optimierte LAB-Version

OKLAB bietet verbesserte perzeptuelle Eigenschaften gegenüber dem traditionellen LAB.

OKLAB Rot-Orange
oklab(0.7 0.1 0.1)
Kleinere Zahlenbereiche als LAB
OKLAB Gelbgrün
oklab(0.8 -0.1 0.1)
Optimiert für Human Vision System
OKLAB Blau-Violett
oklab(0.6 0.05 -0.15)
Bessere Gradients und Übergänge

Display P3 Color Space - Wide Gamut

Display P3 erweitert den sRGB-Farbraum und ermöglicht lebendigere Farben auf modernen Displays.

P3 Rot
color(display-p3 1 0 0)
25% größerer Farbraum als sRGB
P3 Grün
color(display-p3 0 1 0)
Besonders lebendige Grüntöne
P3 Blau
color(display-p3 0 0 1)
Tiefere, sattere Blautöne

P3 vs. sRGB Vergleich

P3: color(display-p3 1 0.3 0.3)
sRGB: rgb(255, 77, 77)

Rec2020 Color Space - Zukunft der Displays

Rec2020 definiert einen noch größeren Farbraum für zukünftige Display-Technologien.

Rec2020 Rot
color(rec2020 1 0 0)
Etwa 70% größer als sRGB
Rec2020 Grün
color(rec2020 0 1 0)
Extrem lebendige Farben
Rec2020 Blau
color(rec2020 0 0 1)
Zukunft der Farbdarstellung
Rec2020 Farben sind nur auf speziellen Wide-Gamut Displays vollständig sichtbar

XYZ Color Space - CIE Standard

XYZ ist der internationale Standard-Farbraum der CIE (Commission Internationale de l'Éclairage).

XYZ Warm
color(xyz 0.4 0.2 0.1)
Basis für Farbwissenschaft
XYZ Neutral
color(xyz 0.2 0.4 0.3)
Geräteunabhängiger Standard
XYZ Kühl
color(xyz 0.1 0.2 0.6)
Referenz für alle anderen Farbräume

Farbverläufe und Interpolation

CSS Color 4 ermöglicht Farbverläufe in verschiedenen Farbräumen mit unterschiedlichen Ergebnissen.

RGB vs. LAB Gradient

linear-gradient(in srgb to right, red, blue) - RGB Interpolation

linear-gradient(in lab to right, red, blue) - LAB Interpolation

OKLCH Gradient mit Hue-Interpolation

linear-gradient(in oklch to right, oklch(0.7 0.15 0), oklch(0.7 0.15 120))
